32 C This program, 'asc2eph', requires (via standard input) an ascii
33 C header file ('header.XXX'), followed by one or more ascii ephemeris
34 C data files ('ascSYYYY.XXX'). All files must have the same ephemeris
35 C number, XXX. Further, the data files must be consecutive in time
36 C with no gaps between them.
37 C
38 C By default, the output ephemeris will span the same interval as the input
39 C text file(s). If you are interested in only a portion of data, set the
40 C below T1 and T2 to the begin and end times of the span you desire. T1
41 C and T2 must be specified in Julian Ephemeris Days (ET).
42 C
43 C A sample sequence of files might be:
44 C
45 C header.405 asc+1920.405 asc+1940.405 asc+1960.405 asc+1980.405
46 C
